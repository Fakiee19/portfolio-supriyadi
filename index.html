Saya Seorang Desain Grafis & Front End Web Developer

Saya seorang Desain Grafis & Front End Web Developer dengan pengalaman bertahun-tahun di dunia Desain Grafis. Saat ini saya adalah Desain Freelancer yang menciptakan karya seni untuk brand pakaian. Sepanjang karir saya, saya beruntung bisa memegang banyak jabatan (Desain Grafis, Front End Web Developer, UI/UX, Konten Kreator, Admin, dan Digital Marketing) di beberapa tempat. Dan Seni Rupa selalu menjadi tulang punggung praktik saya. Saya menyukai pekerjaan ini dan saya suka menciptakan pengalaman baru bagi para penikmat seni.

Saat ini saya berusia 28 tahun, saya sudah memiliki beberapa skill yang dapat diandalkan seperti DESAIN, FRONT END WEB DESIGN, PROGRAM OFFICE, FOTOGRAPHY dan beberapa skill EDITING VIDEO.

Saya merupkan seorang lulusan SMK, dalam TEKNIK KOMPUTER & INFORMATIKA “Rekayasa Perangkat Lunak” di SMK Negri 01 Pabelan. Saya memulai perjalanan profesional saya di beberapa Home Industri sebagai Pengembang Usaha dan kemudian sebagai Desainer Grafis untuk media cetak, web dan sosial media. Memiliki 8+ tahun pengalaman menjadi seorang senior grafis desainer. Mengembangkan ikon perusahaan dan meningkatkan awareness brand sebesar 15%. Ahli dalam mengoperasikan Photoshop, Illustrator, Lightroom, CorelDraw, SketchUp, Premiere, Cap Cut, dan Microsoft Office, dll. Saya juga sedikit memiliki kemampuan sebagai PROGRAMER, bahasa pemprograman yang saya kuasai adalah HTML dan CSS, ini adalah contoh website yang saya buat sendiri.

Selama bertahun-tahun saya beruntung bisa bekerja dengan beberapa profesional paling berbakat di industri ini dan terlibat dalam proyek-proyek luar biasa. Di tahun 2017 saya juga pernah membuat website / blog Pemerintahan Desa dan sampai saat ini saya juga masih mengerjakan beberapa proyek desain disana.
